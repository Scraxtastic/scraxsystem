# Commands

## Server

### Start

`systemctl start scraxserviceServer`

### Stop

`systemctl stop scraxserviceServer`

### Restart

`systemctl restart scraxserviceServer`

### Show

`journalctl -fu scraxserviceServer`

### Status

`systemctl status scraxserviceServer`
